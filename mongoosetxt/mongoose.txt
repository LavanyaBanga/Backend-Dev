Middleware (hooks) are functions that run before or after certain Mongoose operations.

Purposes include:

Pre-save logic:

Example: hashing a password before saving a user.
userSchema.pre('save', async function(next) {
    this.password = await bcrypt.hash(this.password, 10);
    next();
});

Post-processing:

Example: logging or sending a notification after a document is saved.
userSchema.post('save', function(doc) {
    console.log('New user created:', doc);
});
Centralized data handling:
Middleware allows you to reuse logic across multiple operations (save, update, delete).
